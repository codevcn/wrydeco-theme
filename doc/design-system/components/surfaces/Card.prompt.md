# Card

Elevated surface for containing content with shadow depth.

```jsx
<Card elevation={1}>
  <h3>Product Details</h3>
  <p>Premium handcrafted walnut shelf...</p>
</Card>
```

## Variants
- **default** — Cream background (standard)
- **elevated** — White background (premium)

## Elevation Levels
1. Cards, default surfaces
2. Hover state, elevated cards
3. Dropdowns, popovers
4. Modals, fullscreen overlays

Hover automatically increases elevation by 1 level.
