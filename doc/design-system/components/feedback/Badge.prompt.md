# Badge

Small labeled badge for status, tags, or labels.

```jsx
<Badge variant="success">In Stock</Badge>
<Badge variant="accent">New</Badge>
```

## Variants
- **default** — Stone background
- **success** — Sage (success state)
- **warning** — Gold (warning state)
- **error** — Rust (error state)
- **accent** — Clay (highlight)

## Sizes
- **sm** — Small labels
- **md** — Standard
- **lg** — Prominent labels
