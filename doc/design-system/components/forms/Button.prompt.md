# Button

Primary action button for forms and interactions.

```jsx
<Button variant="primary" size="md">
  Explore Collection
</Button>
```

## Variants
- **primary** — Dark, elevated action button
- **secondary** — Stone background, secondary action
- **ghost** — Transparent, outlined style

## Sizes
- **sm** — 32px (compact UI)
- **md** — 40px (standard)
- **lg** — 48px (prominent CTA)

## States
- **Default** — Standard appearance
- **Hover** — Darkens and elevates shadow
- **Disabled** — Reduced opacity, disabled cursor
- **Focus** — Sage outline for keyboard navigation
