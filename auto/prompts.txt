Popup đang mở có 2 side, side trái và side phải, side phải chứa các customization type (ví dụ type "Choose Size"), trong mỗi customization type chứa các customization option (ví dụ QUEEN SIZE - 60"W X 80"L), bạn hãy nhóm các customization option lại thành các bộ customization option khác nhau (ví dụ bộ QUEEN SIZE - 60"W X 80"L & No Need CustomSize, I understand variation & I will check Amazon Messages and respond promptly & No, I will install it myself), bộ nào mà có giá tăng thêm thì ghi nhận luôn giá tăng thêm chính xác vào bộ đó.
bỏ đi type "Note to seller".
cuối cùng tôi cần 1 file json đã nhóm các customization option lại thành các bộ khác nhau.
trong quá trình nhóm thì mặc định bỏ customization option đầu tiên của type Size đi (cứ customiz ation type nào có chứa chữ "size" thì bỏ đi option đầu tiên, chữ size ko phân biệt chữ hoa chữ thường).
